Proyecto Mayorista - Alan Helados

Descripción del Proyecto
Sitio web ha sido desarrollado para  la Pre-Entrega obligatoria para el curso Front-End JS de Talento Tech.
Consiste en una plataforma institucional adaptada de forma estática y responsiva para una distribuidora "Alan", enfocada en la venta mayorista de helados
y congelados.

Tecnologías Aplicadas
- HTML5: Uso estricto de etiquetas estructuradas <header>, nav, main, section, article y footer.
- CSS3 : Uso de cajas Box Model, uso de fuentes tipográficas externas vía Google Fonts (Montserrat y Open Sans), fondos mediante degradados dinámicos  
         y pseudo-clases interactivos (`:hover`, `:focus`).
- Diseño Responsivo: Uso de Flexbox, Grid Layout y adaptabilidad a móviles con Media Queries con un corte menos a 795px

Formulario (Formspree)
Para dotar al formulario de contacto de funcionalidad completa sin depender de código de servidor avanzado (backend):
se utilizó la plataforma externa Formspree, de acuerdo a lo visto en las practicas de los jueves. 
1. Se estructuraron los campos críticos con sus respectivos atributos name y etiquetas label.
2. Se configuró el envío mediante el método `POST` apuntando hacia el endpoint seguro provisto por la plataforma.
